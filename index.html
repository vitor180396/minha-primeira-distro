Por que trocar Windows pelo Linux?

Vantagens de utilizar Linux

Gratuito

Você não precisa ativar o Linux como em outros sistemas. É totalmente gratuito.

Seguro

Construido apartir de uma comunidade enorme, você não precisa se preocupar com nada.

Fácil

Mesmo que você não saiba nada sobre computadores, sempre tem um Linux para você.

Personalizável

O Linux não te esconde nada, personalização nas suas mãos é só uma questão de tempo
